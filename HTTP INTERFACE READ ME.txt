== DETAIL ==
 === List all devices===
    `zVirtualScene?cmd=ListDevices`
 
 === List all Scenes=== 
    `zVirtualScene?cmd=ListScenes`

 === Repoll ZWave Devices===
    `zVirtualScene?cmd=RepollDevices`    
 
 === Run Scene=== 
    `zVirtualScene?cmd=RunScene&Scene=0`
      * Scene = any valid seen ID

 === Control BinarySwitch===
    `zVirtualScene?cmd=BinaryPowerSwitch&node=2&state=ON`
	  * node  = any valid MultilevelPowerSwitch node
      * state =  ON | OFF

 === Control MultilevelPowerSwitch===
    `zVirtualScene?cmd=MultilevelPowerSwitch&node=2&level=0`
      * node  = any valid MultilevelPowerSwitch node
      * level = any valid lievel (0-99)
	
 === Control Thermostat===	
    `zVirtualScene?cmd=ThermoStat&node=7&HeatCoolMode=-1&FanMode=-1&EngeryMode=-1&HeatPoint=-1&CoolPoint=-1`
      
        * node = any valid GeneralThermostat node ID

        * HeatCoolMode 
          * DoNotChange = -1 
          * Off = 0
          * Heat = 1
          * Cool = 2
          * Auto = 3
          * AuxiliaryOrEmergencyHeat = 4
          * Resume = 5
          * FanOnly = 6
          * Furnace = 7
          * DryAir = 8
          * MoistAir = 9
          * AutoChangeover = 10
          * HeatEcon = 11
          * CoolEcon = 12

        * FanMode
          * DoNotChange = -1
          * AutoLow = 0
          * OnLow = 1
          * AutoHigh = 2
          * OnHigh = 3

        * EnergyMode
          * DoNotChange = -1
          * EnergySavingMode = 0
          * ComfortMode = 255
	
        * HeatPoint
          * DoNotChange = -1
          * Valid Temperature in F

        * CoolPoint
          * DoNotChange = -1 
          * Temperature in F
