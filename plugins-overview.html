Plug-in Overview

LightSwitch Server Plug-in

Control your devices from you Android or iPhone using the LightSwitch smartphone app.

Speech Plug-in

Announces events such as 'Bedroom Light changed to 99 from 0' audibly on your PC speaker.

NOAA Plug-in

Activate scenes at sunrise and sunset.

WebApi2 Plug-in

Interact with nearly every aspect of zVirtualScenes using a RESTful API. This uses Microsoft Web API and oData4. Supported Odata operations can be found here.

Thing Speak Plug-in

This plug-in will upload device changes to https://thingspeak.com.
